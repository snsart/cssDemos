# cssDemos

### css编写思考流程

1. 考虑元素是块级元素还是行内元素
2. 考虑元素的定位：相对定位还是绝对定位
3. 考虑元素left top margin padding等属性
4. 考虑元素大小颜色等属性

若元素是行内元素，考虑line-height vertical-align text-align 以及继承等属性
